Как пользоватсья программой:
Копирование не работает, windows не дает доступ к файлам
Копирование копирует файл из одной папки показанной в листбоксе в другую.
Создание открывает отдельное окно в котором нужно ввести имя папки и она создастся
Удаление удаляет выбранные файл
В окне информации выводится информация о файле или каталоге при нажатии кнопки информация так же появляется полный путь до файла
Для обновления информации о файлах нажмите Обновить
Чтобы открыть папку нажмите открыть и в окне выведится список подкаталогов
Кнопка выхода выход из приложения
При нажатии кнопки F1 открывается информация о приложении
